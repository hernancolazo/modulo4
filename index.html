Al abrir la terminal podemos comenzar a crear carpetas o archivos donde nosotros queramos. Para ello hay comandos, un ejemplo de crear una carpeta en el escritorio y luego entrar a esa carpeta y crear archivos es el siguiente:

MacBook-Pro:~ macbook$ cd “desktop” esto nos hace posicionarse dentro del escritorio

MacBook-Pro:~ macbook$ mkdir “nombre de la carpeta”

MacBook-Pro:~ macbook$ cd “nombre de la carpeta” esto hace que entremos a la carpeta

MacBook-Pro:~ macbook$ touch “index.html” con esto creamos el archivo, puede ser .ccs





Estos son los pasos para crear dentro de la terminal. Ahora cerramos la terminal y sobre el VSC arrastramos la carpeta para que se abra en el mismo. Acá trabajamos en html y css normalmente y cuando queremos mandar a un repositorio en git, hacemos los siguientes pasos en la terminal:

MacBook-Pro:~ macbook$ git init

MacBook-Pro:~ macbook$ git add . al poner el punto toma “todos los archivos” o bien se puede poner

MacBook-Pro:~ macbook$ git add index.html

MacBook-Pro:~ macbook$ git add estilos.css

MacBook-Pro:~ macbook$ git commit -m “mensaje que queramos dejar” [master (root-commit) 2c24aa5] aca te muestra el mensaje 2 files changed, 54 insertions(+) create mode 100644 estilos.ccs create mode 100644 index.html



En este paso, lo vamos a vincular al repo de git, creamos el repo SIN README y seguimos las indicaciones:

MacBook-Pro-de-Hernan:~ macbook$ git remote add origin https://github.com/hernancolazo/aaaa.git

MacBook-Pro:~ macbook$ git branch -M main

MacBook-Pro:~ macbook$ git push origin main

De acá en adelante, trabajamos y todo lo que hacemos para subir el repositorio es:

MacBook-Pro:~ macbook$ git init

MacBook-Pro:~ macbook$ git add .

MacBook-Pro:~ macbook$ git commit -m “mensaje que queramos dejar”

MacBook-Pro:~ macbook$ git push origin main
